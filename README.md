# 🥶 아이스 브레이킹 게임 서비스 | 얼음땡 🥶

![썸네일](https://github.com/gumsu/ice-ttaeng-server/assets/49813916/919712aa-c3eb-4496-b322-de79a6c4cd1a)

## 🤖 꽁꽁 얼어붙은 어색함을 부숴줄 아이스 브레이킹 도구 모음 🤖
> - 자유 퀴즈를 만들고 정답을 맞출 수 있는 스피드게임
> - 참여자의 기분을 점수로 확인할 수 있는 기분체크인
> - 서로에게 감사한 일을 전달하는 감사서클
> - 소규모로 그룹을 구성하는 미니네트워킹

## 🚀 프로젝트 사용 기술
- Java 17
- Spring Boot 3.0.6
- PostgreSQL 14
